HalfThr - Iterative Half Thresholding Algorithm

Description

The function aims to solve l_{2,\frac{1}{2}} regularized least squares.

Usage

HalfThr(A, B, X1, maxIter, s)

Arguments

A

Gene expression data of transcriptome factors (i.e. feature matrix in machine learning). The dimension of A is n * t.

B

Gene expression data of target genes (i.e. observation matrix in machine learning). The dimension of B is m * t.

X1

Gene expression data of Chromatin immunoprecipitation or other matrix (i.e. initial iterative point in machine learning). The dimension of InitialX is m * n.

maxIter

Maximum iteration used in computation.

s

Group sparsity level

Details

The HalfThr function aims to solve the problem:

\min \|Ax-b\|^2 + λ \|x\|_{2,\frac{1}{2}}

to obtain s-group sparse solution.
